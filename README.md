# Excel Macros Project

## Описание
Репозиторий для хранения и развития VBA-макросов, используемых для автоматизации обработки реестров и записи накладных в целевые таблицы.

## Структура
- `src/` — исходные `.bas` файлы макросов.
- `docs/` — документация и история изменений.
- `tests/` — тестовые Excel-файлы для проверки работы макросов.
- `.gitignore` — исключает временные файлы и лишние артефакты.
- `LICENSE` — лицензия проекта.

## Как использовать
1. Импортируйте `.bas` файл из папки `src/` в Excel VBA Editor.
2. Запустите макрос `RecordOrdersNumberInTable`.
3. Для тестирования используйте файлы из папки `tests/`.

## Версионирование
Все изменения фиксируются в `CHANGELOG.md`.  
Основной рабочий макрос хранится в `src/RecordOrders.bas`.

## Требования
- Excel 2016 или новее.
- Включённая поддержка VBA.
