# Changelog

Все заметные изменения в этом проекте будут документироваться в этом файле.

## [0.1.0] - 2026-03-28

### Added

- Создан репозиторий по структуре (`src/`, `docs/`, `tests/`).
- Экспортирован базовый макрос `RecordOrdersNumberInTable` в файл `src/RecordOrders.bas`.
- Добавлен `.gitignore` для исключения временных Excel-файлов и артефактов.
- Подготовлен стартовый `README.md` с описанием проекта и инструкцией по запуску.
- Заложен файл `CHANGELOG.md` для ведения истории изменений.
  v0.1 — Начало работы
- Создана базовая процедура CheckOrCreateSubRows.
- Реализована вставка строк под шапкой и заполнение номеров накладных.
- Добавлена логика пустой строки в конце подшапки.
  v0.2 — Исправление ошибок группировки
- Добавлена проверка OutlineLevel перед вызовом .ShowDetail.
- Ошибка Run-time error 1004 устранена.
- Вставка строк и группировка стали устойчивыми.
  v0.3 — Конкатенация номеров в шапке
- Шапка теперь формируется как строка с номерами накладных через ;.
- Подшапка остаётся списком отдельных накладных.
  v0.4 — Гиперссылки в подшапке
- В подшапке номера заменены на формулы HYPERLINK.
- Добавлена переменная basePath для относительного пути.
- Каждая накладная теперь кликабельна.
  v0.5 — Глобальная константа
- basePath вынесена в глобальную константу Public Const basePath As String.
- Упрощён вызов процедуры: больше не нужно передавать путь параметром.
  v0.6 — Тестирование на рабочем файле
- Добавлена проверка существования файла перед Workbooks.Open.
- Устранена ошибка Method 'Open' of object 'Workbooks' failed.
- Макрос успешно протестирован на реальных данных.
